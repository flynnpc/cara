# Cara
Data processing tool for Cara

## Steps for using on a Mac/Linux

- Open a terminal and navigate to the folder you cloned the Cara repository to, `cd /path/to/cara`
- In the same terminal,run `python3 activate_cara.py` if this is the first time using Cara in this folder to install everything.
- Once the terminal reads "Installation complete!" you can activate the virtual environment `source venv/bin/activate`. If you get an error message, call me :).
- Now that everything is installed and your virtual environment is activated you can run the `python3 transfer_data.py` script. Just follow the prompts in the terminal and reach out if you have any questions.

## Steps for using on Windows

- Open a command prompt and navigate to the folder you cloned the Cara repository to, `cd /path/to/cara`
- In the same command prompt,run `python activate_cara.py` (Or whatever Python command is in your path). If this is the first time using Cara in this folder it will install everything.
- Once the command prompt reads "Installation complete!" you can activate the virtual environment ` venv\Scripts\activate.bat`. If you get an error message, call me :).
- Now that everything is installed and your virtual environment is activated you can run the `python transfer_data.py` script. Just follow the prompts in the terminal and reach out if you have any questions.
